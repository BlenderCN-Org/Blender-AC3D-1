Blender-AC3D by majic79

AC3D (.ac) import/export for Blender 2.5

A small project to bring AC3D import/export functionality into Blender 2.5

Install

Copy the io_scene_ac3d folder and its contents into the blender/2.5X/addons/scripts folder Run blender, enable the addon from user preferences->import-export (save as default to make it always available) Import and export to your hearts content!
